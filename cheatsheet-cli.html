CLI 指令小抄

終端機最常用的指令，一頁帶著走 —— 不確定就翻它
導航：我在哪、有什麼
pwd現在在哪個資料夾（印出完整路徑）
ls列出當前資料夾裡有什麼
ls -la列出全部（含隱藏檔）+ 詳細資訊
cd 資料夾進入某個資料夾
cd ..回上一層
cd ~回到家目錄（你的大本營）
建立：生出新東西
mkdir 名稱新建一個資料夾
touch 檔名新建一個空白檔案
echo "字"把文字印到畫面上
看內容：檔案裡裝什麼
cat 檔名一次印出整個檔案
head 檔名只看前 10 行
head -n 20 檔名看前 20 行
整理：複製、搬移、刪除
cp 來源 目的複製檔案
cp -r 來源 目的複製整個資料夾
mv 來源 目的搬移，或改名
rm 檔名⚠ 刪檔案，不進垃圾桶
rm -r 資料夾⚠ 刪整個資料夾，極危險
搜尋與組合技
grep "字" 檔名在檔案裡找含關鍵字的行
指令 | grep "字"把前一個指令的輸出餵給 grep 過濾
指令 > 檔名把輸出寫進檔案（覆蓋原內容）
指令 >> 檔名把輸出附加到檔案結尾
求救與省力技
man 指令看某指令的說明書（按 q 離開）
clear清空畫面，重新開始
↑ / ↓翻找剛剛打過的指令，不用重打
Tab自動補全檔名／指令，少打字少打錯

保命提醒：rm 沒有垃圾桶，打下去就真的消失。動手前先用 pwd 和 ls 確認你在對的地方、刪對的東西。

提示字元：Mac / Linux 是 $，Windows PowerShell 是 PS>，打指令時不用打它。
少數指令兩平台寫法略有不同，以課堂投影片雙色標示為準。

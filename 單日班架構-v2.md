# CLI、Git 與 Agentic AI 實作 — 單日班架構 v2

> 2026-06-05 更新：開場拉長 10-15min（自我介紹 + AI 互動），Unit 1 從 90→105min，午餐 60→55min，後續不變。

## 設計變更摘要（v1 → v2）

| 項目 | v1 | v2 | 變更理由 |
|------|----|----|---------|
| 課程定位 | CLI 是門票，Git 是基礎，AI 是加速器 | 同，但 Git 降為「橋樑」不為「基礎」 | 用戶確認：Git 重在知道為什麼重要，不是操作深度 |
| 節奏單元 | 不規則（30/150/120/75/45min） | 統一 90min 單元 × 4 + 收尾 35min | 注意力週期 + 微休息設計 |
| 上午內容 | CLI 3h | CLI 3.25h（純 CLI，不加 Git） | 2026-06-05 改：第一堂需自我介紹 + AI 互動，Unit 1 拉長 15min |
| Git 段落 | 2h 正課（含 branch/merge 實作） | 1.5h 概念 + 基本操作 + branch demo | Git 輕量化，branch 用 demo 帶過 |
| AI 時段 | 1.25h（最末段） | 1.25h（下午第二節，接在 Git 後） | AI 不再擠在最後 |
| branch/merge/conflict | 正課教學（每人實作） | 講師 demo 快速帶過 | 用戶確認：demo 即可 |
| 講解:實作比 | ~65:35 | 全天 ~66:34（CLI 偏講解） | 2026-06-05 改：開場拉長 15min，講解比微升 |
| 微休息 | 無 | 每 25min 插 90sec | 維持正確率優勢 20.6% |
| 安全網 | 彈性調整 | 紅綠便利貼 + Codespaces 備案 + 可刪減段落 | 防禦性設計 |
| Claude Code 安裝 | 不明確 | 課前預裝 + Codespaces 備案 | 用戶確認 |

## 課程設計理念

### 三層能力架構

| 層次 | 能力 | 定位 | 學員帶走什麼 |
|------|------|------|-------------|
| CLI 操作 | 命令列基礎 | 門票 | 不怕黑畫面，能獨立操作檔案和目錄 |
| Git 版本控制 | 版本追蹤 | 橋樑 | 知道為什麼要版控、會 add/commit/push |
| Agentic AI 協作 | AI 指揮 | 目的地 | 能用自然語言讓 AI 在 CLI 做事，會審查 AI 輸出 |

### v2 設計原則

- **講解先於實作**：零基礎要先建立心智模型，再動手（Peyton 4-Step：demo→解說→跟做→獨立）
- **90 分鐘單元制**：每個單元聚焦，單元間強制休息
- **實作段硬上限**：每段實作有明確時間，到點就收
- **Git 輕量化**：重「為什麼需要版控」輕「怎麼操作所有指令」，branch/merge 用 demo
- **可刪減段落明確標注**：時間不夠時有安全砍掉順序

```bash
# Step 1: 建專案資料夾
mkdir my-project
cd my-project

# Step 2: 建自我介紹檔案
touch intro.txt

# Step 3: 寫入內容
echo "我叫 ___，今天學 CLI" > intro.txt

# Step 4: 確認內容
cat intro.txt

# Step 5: 搜尋關鍵字
grep "CLI" intro.txt

# Step 6: 回到上層確認結構
cd ..
ls my-project/
```

### 帶走的東西

- [ ] CLI 指令小抄（雙面）
- [ ] Git 指令小抄（雙面）——只列五條命 + 常用 flag
- [ ] 三層能力架構圖
- [ ] 延伸資源清單（git-scm.com、learngitbranching.js.org、Claude Code docs）
- [ ] 「為什麼要版控」一頁摘要

### 課後 3 天內群組互動

- 丟小挑戰：「用 git 建一個 repo，push 一個 commit，截圖到群組」
- 保持熱度，收集真實使用案例

### 進階班銜接

單日班 = CLI + Git 概念 + Agentic AI 入門
進階班 = Git 深入（branch/merge/衝突實作）+ AI 工作流整合（Claude Code 深入、MCP/Skills、n8n 自動化、CI/CD）

課程中埋的 hook：
- 開場三層地圖：「今天把三層地基打好」
- Git 段落：「branch 和衝突今天用看的，進階班你會自己操作」
- AI 實作段落：「進階班會教你讓 AI 自己跑整個工作流」
- 收尾：「三層能力你都有了，下一步是把它們串成自動化流程」

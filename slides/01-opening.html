動手前導

裝好 opencode
Windows 先 Mac 後

現場估計大家都還沒裝。Windows 需先裝 Node.js（到 nodejs.org 下載 LTS，雙擊 .msi 安裝後重開終端），再用 npm 裝 opencode。Mac 用官方一行腳本，免 Node。裝完輸入 opencode 啟動。

# Windows — PowerShell
# 1. 先裝 Node.js：到 nodejs.org 下載 LTS，雙擊 .msi
# 裝完重開 PowerShell，確認版本：
PS> node -v
# 2. 裝 opencode
PS> npm install -g opencode-ai
# 沒 Node 也可：scoop install opencode
# Mac — Terminal（免 Node，官方一行裝好）
$ curl -fsSL https://opencode.ai/install | bash
# 或 Homebrew
$ brew install anomalyco/tap/opencode

動手前導

打開黑畫面
找到作品位

先學五個生存指令。它們幫你知道自己在哪、進資料夾、建資料夾、看檔案、建新檔。

$ pwd # 看自己在哪
$ cd Desktop # 進桌面
$ mkdir my-page # 建資料夾
$ cd my-page # 進入專案
$ ls # 看檔案清單
$ touch index.html # Mac 建新檔
$ New-Item index.html # Windows 建新檔

MINDSET

從點滑鼠到下指令

命令列不是工程師的專利。
它是一種跟電腦溝通的方式 — 更快、更精確、更強大。

# 點滑鼠：找到資料夾 → 右鍵 → 新增資料夾 → 打名字
# 下指令：
$ mkdir my-project
# 點滑鼠：找終端機 → 裝 Git → 設定 → 建 repo
# 下指令：
$ git init && git add . && git commit -m "start"

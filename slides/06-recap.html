把八小時走過的路再走一次。五個單元、三層能力、一個本機作品。

今日流程

五個單元走一遍

從打開終端機到本機作品，每一步都有可見的成果。

1

UNIT 1 開場 — 不怕黑畫面

打開終端機，pwd / ls 看自己在哪，建立 my-page 與 index.html。

2

UNIT 2 第一個 HTML — opencode 動手

跟 opencode 說一句話，AI 自己寫入 index.html，瀏覽器打開就看到頁面。

3

UNIT 3 改版 — 從能看到像作品

好指令 vs 壞指令，學會把需求講成主題、區塊、風格三要素。

4

UNIT 4 自由實作 — 把作品做完

名片頁、商家頁、活動頁三選一，用 opencode 把選定主題做出來。

5

UNIT 5 回顧 — 打開你的 index.html

本機瀏覽器展示成果，互相看彼此的作品。

三層能力

今天累積的能力

不是學工程師技能，是把這三層串起來：說清楚 → 看懂 → 做出來。

01

AI 對話

把想法說清楚，讓 opencode 直接改檔，不用複製貼上。

02

terminal 看懂

找檔案、進資料夾、開檔案，看懂 AI 到底做了什麼。

03

HTML 作品

本機網頁產出，瀏覽器打開 index.html 就是你的成果。

今天會的指令

這幾個就夠了

不用背指令表。這幾個是為了「知道作品在哪、打開來看」而存在的。

# 知道自己在哪
$ pwd
$ ls
# 進資料夾、建資料夾
$ cd Desktop/my-page
$ mkdir my-page
# 建檔、打開看成果
$ touch index.html
open index.html # Mac
start index.html # Windows
# 然後交給 opencode 改檔
$ opencode

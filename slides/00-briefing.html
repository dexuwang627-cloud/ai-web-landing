BOTH

Mac 和 Windows 都教

投影片會同時標示兩邊的指令。
橘色 = Mac ｜ 藍色 = Windows

Mac → Terminal / iTerm2
$ pwd
$ ls -la
$ touch file.txt
Windows → PowerShell / Windows Terminal
PS> pwd
PS> ls
PS> New-Item file.txt
PREP

上課前準備

1

確認終端機能用

2

3

準備 AI 工具帳號

4

安裝 Claude Code（建議預裝）

npm install -g @anthropic-ai/claude ｜ 課堂上也會協助安裝

安裝前確認 Node.js 已裝好：終端機輸入 node -v，有顯示版本號即可

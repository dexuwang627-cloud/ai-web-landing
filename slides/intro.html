PRE-CLASS SETUP

不用先準備，現場一起裝

明天的課會帶你從零開始安裝。即使電腦是全新的，也只要跟著步驟做就能完成。

01

現場一步一步帶

講師會同步示範 Mac 與 Windows 的安裝流程，照著做即可。

02

不需要先裝任何軟體

明天來之前不必預先下載，我們會在課堂上確認每個人都能使用。

03

有問題隨時舉手

如果指令跑不出來，我們會一起看畫面、找原因、解決它。

Mac INSTALL

Mac：一行指令就裝好

官方一鍵安裝不需要 Homebrew 或 npm；macOS 已內建 curl 與 unzip，什麼都不用先裝。

curl -fsSL https://opencode.ai/install | bash
1

偵測 Mac 晶片並下載

自動判斷 Apple Silicon 或 Intel，從 GitHub Releases 下載對應的 zip。

2

解壓並加入 PATH

把 opencode 放到 ~/.opencode/bin/opencode，並自動設定環境變數。

3

驗證安裝

在新終端機輸入 opencode --version，看到版本號即成功。

若已有 Homebrew，也可用 brew install anomalyco/tap/opencode；官方腳本會自動保持最新。如果自動安裝卡住，可手動下載對應 zip：

Apple Silicon
https://github.com/anomalyco/opencode/releases/latest/download/opencode-darwin-arm64.zip
Intel
https://github.com/anomalyco/opencode/releases/latest/download/opencode-darwin-x64.zip

Windows INSTALL

Windows：推薦用 npm 安裝

課堂上最穩定的路徑是先裝 Node.js，再用 npm 全域安裝 opencode-ai。

Node.js 是「讓 JavaScript 在你電腦上執行的環境」；npm 是「下載工具包的程式」，安裝 Node.js 時會自動附帶。我們先裝 Node.js，主要目的就是取得 npm。

1

下載 Node.js 安裝檔

前往 nodejs.org，點擊 Get Node.js 按鈕進入下載頁，下載 Windows 的 .msi 安裝檔（預設 LTS）。

2

執行安裝程式

雙擊下載的安裝檔，一路按「下一步」，使用預設安裝路徑即可；npm 會自動一起裝好。

3

確認安裝成功

開啟終端機（PowerShell 或 Git Bash），輸入以下指令，看到版本號就代表沒問題：

node -v npm -v
4

安裝 opencode

在終端機貼上這行指令，全域安裝 opencode-ai：

npm install -g opencode-ai

小提示：如果輸入 npm 後出現「不是可識別的命令」，請先關閉並重新開啟終端機再試一次。

若 npm 無法使用，可改用官方壓縮檔：

https://github.com/anomalyco/opencode/releases/latest/download/opencode-windows-x64.zip
1

下載並解壓縮

下載 zip 後，把資料夾解壓到你想放的位置。

2

加入系統路徑 PATH

把解壓後的資料夾路徑加入 PATH，之後就能在任何終端機直接使用 opencode。

FIRST RUN

第一次啟動：登入並加入 API 金鑰

裝好後第一次執行，需要讓 opencode 知道你是誰。跟著畫面提示做即可。

1

啟動登入

在終端機輸入 opencode，然後在對話框內輸入 /connect，或執行 opencode auth login。

2

開啟授權網頁

畫面會顯示網址 opencode.ai/auth。用瀏覽器打開它、登入並綁定付款後，網頁會產生一組 API 金鑰。

3

登入並綁定付款

用 Google 或 GitHub 登入後，加入一張信用卡（可從最低額度開始）。

4

貼上 API 金鑰

網頁會產生一組金鑰，複製後貼回終端機，按 Enter 就完成了。

?

卡住時試試看

如果瀏覽器授權後終端機沒反應，先確認網路連線，然後重新輸入 opencode auth login 再走一次流程。

VERIFY

確認 opencode 可以正常運作

點擊下方項目標記完成。三道檢查都通過，就可以開始跟 AI 協作。

# 檢查版本 opencode --version # 進入練習資料夾 cd Desktop/my-page # 啟動 opencode opencode
0/3
終端機顯示 opencode 版本號
成功進入 Desktop/my-page 資料夾
輸入 opencode 後出現對話提示

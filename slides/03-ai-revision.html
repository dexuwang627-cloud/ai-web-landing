帶你走

三步逛完資源

樣板範例 · 逛

別人做過的頁面

挑一個結構喜歡的，記下它的區塊順序。點卡片連過去看。

名片頁

Carrd

carrd.co

單頁網站範本平台，極簡置中結構，適合模仿個人名片頁。

 商家頁

Webflow Templates

webflow.com/templates

分類範本平台，含商家與服務業版型，適合模仿商家介紹頁。

 活動頁

One Page Love

onepagelove.com

單頁網站設計 gallery，含活動頁範例，適合模仿活動版型。

 綜合

Squarespace Templates

squarespace.com/templates

官方範本展示，商家/活動/作品集分類清楚，可挑封面版型。

任務一

挑一個你想做的版型

1

打開上一頁任一個樣板站

Carrd、Webflow、One Page Love 都行，挑順眼的進去逛。

2

找一個你最想做的版型

不用懂技術，看感覺：這個排法我喜歡。

3

記下它的區塊順序

從上到下唸出來，例如：大標 → 介紹 → 三欄卡片 → 聯絡。

# 我的發現（任務一產出）
$ 版型參考：Webflow 的咖啡店範本
$ 區塊順序：店名大標 → 招牌語 → 三項主打 → 地圖聯絡

設計靈感 · 記

找你喜歡的風格

看到喜歡的配色或排版，截圖或用三個詞描述，等等給 AI。

獎項站

Awwwards

awwwards.com

網頁設計獎項站，按分類瀏覽得獎網站找風格靈感。

 著陸頁

Land-book

land-book.com

著陸頁設計 gallery，適合找單頁版型與排版靈感。

 目錄站

Siteinspire

siteinspire.com

網頁設計目錄，可按 tag 篩選風格找版型。

 視覺作品

Dribbble

dribbble.com

設計師作品社群，看配色與介面練習的視覺靈感。

任務二

截圖一個你喜歡的風格

1

打開上一頁任一個靈感站

Awwwards、Land-book、Siteinspire 都行。

2

找到一個你喜歡的配色或排版

截圖存下來，等等可以貼給 opencode 看。

3

用三個詞描述它

例如：深色背景、留白多、圓角卡片。

# 我的發現（任務二產出）
$ 風格截圖：已存到桌面
$ 三個詞：深色、留白多、圓角

AI 工具

回家繼續用的幫手

opencode 是主工具，其他是備援或輔助構思。任一個能產生 HTML 放進 index.html 就算過關。

主工具

opencode

opencode.ai

本機 AI 直接改檔，今天的主工具。Mac / Windows / Linux。

 備援

ChatGPT

chatgpt.com

構思內容、生成網頁文案。網頁版手動貼 HTML 是最後防線。

 備援

Claude

claude.ai

輔助構思與除錯，定位同 ChatGPT。網頁介面不分平台。

 備援

Codex CLI

github.com/openai/codex

OpenAI 本機 coding agent，與 opencode 同類定位，用 ChatGPT 方案額度。

任務三

把發現寫成一句 prompt

把任務一的版型加任務二的風格，套進這個模板，下午實作直接用。

# prompt 模板（把 [ ] 換成你的發現）
我想做像 [Carrd 那種置中] 的個人名片頁，
風格 [深色、留白多、圓角]，
區塊順序 [大標 → 介紹 → 三欄技能 → 聯絡]。
直接寫入 index.html。
# 我的 prompt（填完即可帶進下一階段）
我想做像 ____________ 的 ____________ 頁，
風格 ____________，區塊順序 ____________。

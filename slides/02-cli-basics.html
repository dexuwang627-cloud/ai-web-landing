我在哪？pwd

Print Working Directory — 告訴你現在在哪個資料夾

Mac / Linux

cd = Change Directory = 換資料夾

TRY IT

# 5. 再確認一次
$ pwd

FILE OPS

複製、搬移、刪除

# 複製檔案
$ cp index.html backup.html
# 搬移 / 改名
$ mv backup.html archives/
# 刪除（小心！沒有資源回收桶）
$ rm archives/backup.html
$ rm -r old-folder/

rm 沒有「還原」。刪了就是刪了。

習慣：刪之前先用 ls 確認，或加 -i 參數（互動確認）

TRY IT

READ

看檔案內容

# 看整個檔案
$ cat README.md
# 看前 10 行

# 搜尋關鍵字

# 看即時更新的 log（超實用）

cat印出整個檔案內容 head -N只看前 N 行 grep "字"在檔案裡搜尋關鍵字 tail -f即時監看（AI 工具常用）
POWER MOVE

管線 | 與重導向 >

把一個指令的輸出，變成下一個指令的輸入。
這就是命令列最強大的地方。

# 找出所有 .js 檔案
$ ls -R | grep ".js"
# 數有多少個檔案
$ ls | wc -l
# 把結果存到檔案
$ ls -la > file-list.txt

| = 管線（pipe）｜> = 重導向（redirect）

TRY IT

動手：管線與搜尋

# 在 cli-practice 裡建一個有內容的檔案
$ echo "Hello CLI" > hello.txt
$ echo "Learning grep" >> hello.txt
# 搜尋關鍵字
$ grep "CLI" hello.txt
Hello CLI
# 數有幾行
$ cat hello.txt | wc -l
2

挑戰：用 ls | grep .html 只列出 .html 檔案。

ENV

環境變數與 PATH

環境變數就像電腦的「記憶」— 告訴程式去哪找東西。
PATH 是最重要的環境變數：電腦去哪找指令。

Mac / Linux
$ echo $PATH
/usr/local/bin:/usr/bin:/bin
$ export MY_API_KEY="sk-xxx"
$ echo $MY_API_KEY
sk-xxx
Windows
PS> $env:PATH
C:\Windows\system32;C:\Python
PS> $env:MY_API_KEY = "sk-xxx"
PS> $env:MY_API_KEY
sk-xxx

設 API Key 的時候會用到。AI 工具很多需要環境變數。

DAILY

WATCH OUT

新手常見地雷

!

rm 刪了就沒了

沒有資源回收桶。刪之前用 ls 確認，或加 -i

!

忘記 commit 就 push

git add → commit → push，三步缺一不可

?

AI 幻覺

AI 可能生成不存在的指令。看不懂就查，不要盲抄

?

環境變數外洩

API Key 放 .env，不要寫進程式碼或 commit

TIPS

持續練習的方法

01

每天用一次 CLI

cd 到專案、ls 看檔案、git status — 就這三個就夠了

02

每個改動都 commit

不要累積。完成一個功能就 commit，訊息寫清楚

03

用 AI 學 CLI

看不懂指令？問 AI 解釋。把 AI 當老師，不是當打字機

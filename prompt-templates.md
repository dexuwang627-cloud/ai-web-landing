# opencode / AI 對話範本 — 三主題本機 HTML

> 配合《CLI 與 AI HTML 實作 — 單日班架構 v2》使用。
> 這份範本給學員在 opencode（本機 AI CLI）裡貼上直接用，也適用於 ChatGPT Codex 桌面版、Claude Cowork、ChatGPT / Claude 網頁版。
> 三主題：個人名片頁、商家介紹頁、活動頁。每主題各提供四段指令：初始產生、版面改版、風格改版、手機版調整。

---

## 使用方式

1. 先在終端機進到自己的專案資料夾：
   - Mac：`cd Desktop/my-page`
   - Windows：`cd Desktop\my-page`
2. 確認 `index.html` 在資料夾裡：`ls`
3. 啟動 opencode，把下面指令整段貼進對話框，按 Enter。
4. 改完後在終端機開啟看成果：
   - Mac：`open index.html`
   - Windows：`start index.html`

指令裡用 `[中括號]` 標起來的部分，換成你自己的資料再送出。

## 主題一：個人名片頁

### 1-1 初始產生

```
幫我用 opencode 直接改這個資料夾裡的 index.html，做一個「個人名片頁」，單頁 HTML，內建 CSS，不要外部檔案。

內容：
- 名字：[王小明]
- 一句話自我介紹：[喜歡用數據說故事的行銷人]
- 三個技能卡片：[內容企劃]、[數據分析]、[簡報設計]
- 聯絡方式：[Email、LinkedIn 連結]

要求：
- 桌機版為主，置中排版。
- 顏色用低飽和、乾淨的配色。
- 字體用系統預設中文字體就好，不需要載外部字體。
- 產生後直接寫入 index.html，並告訴我改了什麼。
```

### 1-2 版面改版

```
把現在的 index.html 改成有四個區塊的版面，由上到下：

1. Hero 區：名字 + 一句話介紹，占滿第一屏。
2. 關於我：兩到三段文字。
3. 技能與作品：用三欄卡片，每欄一張圖片佔位（用純色色塊即可）+ 標題 + 說明。
4. 聯絡方式：底部一列，放 Email 與社群連結。

直接改 index.html，改完告訴我每個區塊對應到檔案哪一段。
```

### 1-3 風格改版

```
幫我把 index.html 換成「沉穩專業」風格：

- 主色：深藍 #1f3a5f，輔色：淺灰 #f5f5f5，強調色：橘金 #c89b3c。
- 字體放大：標題 36px、內文 18px。
- 區塊之間留白加大到 64px。
- 卡片加圓角 12px 與淡陰影。
- 整體置中，最大寬度 960px。

直接改檔，改完列出你動了哪些 CSS。
```

### 1-4 手機版調整

```
幫 index.html 加上手機版響應式：

- 螢幕寬度小於 768px 時，技能與作品從三欄變一欄直排。
- Hero 區文字縮小到標題 28px、內文 16px。
- 聯絡方式從一列改成直排按鈕，可點擊。
- 加 `<meta name="viewport" content="width=device-width, initial-scale=1">`。

直接改檔，改完告訴我你加了哪一段 media query。
```

## 主題二：商家介紹頁

### 2-1 初始產生

```
幫我用 opencode 直接改 index.html，做一個「商家介紹頁」，單頁 HTML，內建 CSS。

商家資訊：
- 店名：[晨光咖啡館]
- 類型：[獨立咖啡店]
- 一句話招牌語：[用好豆子，開啟你的一天]
- 三項主打商品：[手沖單品]、[拿鐵]、[手作甜點]
- 營業資訊：[地址、電話、營業時間]
- 店鋪特色兩到三句話

要求：
- 桌機版為主，上方一個店名橫幅。
- 配色溫暖（米色、深咖啡色系）。
- 系統預設中文字體。
- 直接寫入 index.html，並說明你做了什麼。
```

### 2-2 版面改版

```
把 index.html 改成五個區塊的店家網頁：

1. 頂部導覽列：店名 + 三個錨點連結（關於、菜單、找我們）。
2. Hero 橫幅：招牌語 + 一張色塊背景圖。
3. 關於店家：兩段文字 + 一張色塊圖。
4. 主打商品：三欄卡片，每欄商品圖（色塊）+ 名稱 + 一句話描述 + 價格。
5. 底部資訊：地址、電話、營業時間、一個地圖色塊。

直接改 index.html，改完告訴我每區塊在檔案裡的位置。
```

### 2-3 風格改版

```
把 index.html 換成「暖色質感小店」風格：

- 主色：深咖啡 #4a2c2a，底色：米色 #f7f1e8，強調色：磚紅 #b5532f。
- 標題用襯線感字體（系統字體 serif 即可），內文用 sans-serif。
- 商品卡片加圓角 16px、淡陰影、hover 時微微上移。
- 區塊之間留白 80px。
- 最大寬度 1000px 置中。

直接改檔，列出你改動的 CSS 樣式。
```

### 2-4 手機版調整

```
幫 index.html 加手機版響應式：

- 螢幕 < 768px 時：
  - 導覽列收合成漢堡選單（純 CSS 點擊展開即可，不用 JavaScript）。
  - 商品三欄變一欄直排。
  - Hero 橫幅文字置中、字級縮小。
  - 底部資訊從兩欄變一欄。
- 加 viewport meta tag。

直接改檔，說明你加了哪些 media query 規則。
```

## 主題三：活動頁

### 3-1 初始產生

```
幫我用 opencode 直接改 index.html，做一個「活動報名頁」，單頁 HTML，內建 CSS。

活動資訊：
- 活動名稱：[週末城市散步：老屋與咖啡]
- 日期時間：[2026/07/20（日）14:00-17:00]
- 地點：[集合地點：捷運 XX 站 1 號出口]
- 活動介紹：兩到三句話
- 三個活動亮點：[老屋導覽]、[咖啡試飲]、[在地故事]
- 報名方式：[Email 或表單連結]

要求：
- 桌機版為主，最上方放大活動名稱與日期。
- 配色清爽（白底 + 一個主色）。
- 系統預設中文字體。
- 直接寫入 index.html，並說明做了什麼。
```

### 3-2 版面改版

```
把 index.html 改成活動頁的標準版面：

1. Hero：活動名稱 + 日期 + 一個「立即報名」按鈕（錨點到報名區）。
2. 活動介紹：兩段文字 + 一張色塊主視覺。
3. 活動亮點：三欄卡片，每欄圖（色塊）+ 標題 + 一句話。
4. 活動資訊：日期、地點、費用、名額，用表格或定義清單呈現。
5. 報名區：報名步驟三點 + 一個報名按鈕。

直接改 index.html，改完告訴我每區塊對應的檔案位置。
```

### 3-3 風格改版

```
把 index.html 換成「清新活動風格」：

- 主色：墨綠 #2f5d50，底色：米白 #fafaf6，強調色：橘 #e08a3c。
- 標題加粗、字距拉開 1px。
- 按鈕：實心主色、圓角 8px、hover 變深一階。
- 卡片：白底、圓角 12px、淡陰影、hover 邊框變主色。
- 區塊留白 72px，最大寬度 920px 置中。

直接改檔，列出你動了哪些 CSS。
```

### 3-4 手機版調整

```
幫 index.html 加手機版響應式：

- 螢幕 < 768px 時：
  - Hero 按鈕變全寬。
  - 活動亮點三欄變一欄。
  - 活動資訊表格改直排，每項一行。
  - 報名按鈕固定在畫面底部（sticky），方便隨時點。
- 加 viewport meta tag。

直接改檔，說明加了哪些 media query 規則。
```

## 共用小提醒

- 改完一段就 `open index.html`（Mac）或 `start index.html`（Windows）看一次成果，不要累積太多改動再看。
- 覺得改壞了，可以跟 AI 說：「回到上一個能看的版本」或「把剛剛那段改回來」。
- 指令越具體，AI 越能一次做對。把顏色、字級、區塊數字寫出來，比說「變好看」有效。
- 看不懂 AI 改了什麼沒關係，可以用 `cat index.html`（Mac / Windows 皆可用 `cat`）捲一下檔案，知道「檔案被改了」就夠。
- opencode 會直接改檔；若用 ChatGPT / Claude 網頁版，記得把 AI 回的程式碼整段複製貼到 `index.html` 裡覆蓋。
